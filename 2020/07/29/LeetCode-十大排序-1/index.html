LeetCode / 十大排序
2020-07-29|LeetCode十大排序算法

优质资源：https://www.cnblogs.com/onepixel/p/7674659.html

比较类排序

  • 排序算法稳定：稳不稳定看交换过程中，相同的元素会不会改变前后顺序

1.冒泡（往后放） / 时间复杂度O(n^2)

  • 直接两两比较，n次遍历，进行进行交换，将最大的元素移至最后。

2.选择（往前放） / 时间复杂度O(n^2)

  • 直接两两比较，进行下标的交换，最后遍历完一遍，将最大的元素和首元素交换。

3.插入 / 时间复杂度O(n^2)

  • 增量为1，选取一个元素，将他元素插入到合适的位置。

4.希尔 / 时间复杂度O(n^1.3~2)

  • 增量为n的插入排序，当元素总体上有序时，插入排序很高效，所以先大致排序整齐。

5.归并排序 / 时间复杂度O(n*log2n)

  • 先分再和，先将元素分成单个个体，然后两两组合，最后合并成一个有序数组

  • 先分再和，双指针一次将元素填入位置

6.快速排序 / 时间复杂度O(nlog2n)_额外空间开销O（nlog2n）

  • 先和再分，以第一个元素为基准数，大的放右，小的放左，然后依次再以基准数排序

  • 注意：这里包含往下取整还是往上，注意

7.堆 / 时间复杂度O(n*log2n)_额外空间O（n）

  • 利用完全二叉树的特点，构造大根堆，然后首位元素交换

  • 父节点：（i - 1）/ 2 （节点从0开始）
  • 左孩子：i*2 + 1
  • 右孩子：i*2 + 2

非比较类排序

8.计数

  • 1-100最快的排数，适用于数字范围不大地数字段。

    • 1.找到数组最大值和最小值
    • 2.count_arr[arr[i] ]++
    • 3.依次将数组填入

9.基数

  • 按照个位、十位、百位依次顺序排序。

10.桶排序

  • 1.首先对元素进行分组 2.组内元素快速排序 3.依次将元素填充
